AssurLink Figma skeleton

Files included:
- tokens.json
- components.json
- frames.json

Import instructions:
1. Create a new Figma file or project.
2. Use a tokens plugin (Figma Tokens) to paste tokens.json to populate your design tokens.
3. Recreate frames using the frames.json as a guide. Use tokens to style components.
4. Create components per components.json.

If you want, provide a Figma project link and I can prepare a .fig export for direct upload.
